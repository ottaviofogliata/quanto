# Quanto Documentation / Documentazione di Quanto

## English
Welcome to the Quanto documentation. Each section introduces a mathematical idea with a gentle explanation followed by a slightly more detailed look.

The sample configuration file `examples/config.yaml` starts with an `asset_class`
field—such as `options` or `stocks`—that selects the instrument type for
experiments.

## Italiano
Benvenuto nella documentazione di Quanto. Ogni sezione introduce un'idea matematica con una spiegazione semplice seguita da uno sguardo un po' più dettagliato.

Il file di configurazione di esempio `examples/config.yaml` inizia con un campo
`asset_class`, ad esempio `options` o `stocks`, che imposta il tipo di
strumento per gli esperimenti.
